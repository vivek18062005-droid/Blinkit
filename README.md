# Blinkit

📊 Blinkit Sales Analysis – Power BI Project
📌 Project Overview

This project presents a comprehensive sales analysis of Blinkit – India’s Last Minute App (formerly Grofers). The dashboard is built using Power BI to analyze sales performance, customer behavior, product trends, and outlet performance.
The objective of this project is to extract meaningful business insights from Blinkit’s sales dataset and support data-driven decision-making.

🎯 Objectives
Analyze total sales performance
Identify top-performing product categories
Understand customer purchasing patterns
Evaluate outlet performance
Compare sales by outlet size, type, and location
Identify opportunities for revenue growth

📂 Dataset Description
The dataset contains transactional and product-level information, including:
Item Identifier
Item Type
Item Fat Content
Item Visibility
Item Weight
Outlet Identifier
Outlet Establishment Year
Outlet Size
Outlet Location Type
Outlet Type
Sales Amount
Rating

🛠 Tools & Technologies Used
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization Techniques

🧹 Data Cleaning & Preparation
The following preprocessing steps were performed:
Replaced missing values in Item Weight
Standardized Item Fat Content categories (e.g., LF → Low Fat)
Removed duplicates
Checked data types and corrected inconsistencies
Created calculated columns and measures using DAX

📊 Dashboard Features
The Power BI dashboard includes:

1️⃣ KPI Cards
Total Sales

Average Rating

2️⃣ Sales Analysis
Sales by Item Type
Sales by Outlet Type
Sales by Outlet Size
Sales by Location Type

3️⃣ Trend Analysis
Sales by Outlet Establishment Year

4️⃣ Filters / Slicers
Outlet Location Type
Outlet Size
Item Type

📈 Key Insights
Supermarket Type outlets generate the highest sales.
Medium-sized outlets contribute more revenue compared to small ones.
Tier 3 locations show strong sales performance.
Certain product categories dominate total revenue contribution.
